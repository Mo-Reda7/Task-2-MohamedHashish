# Iris Classification using KNN

## Project Description

This project implements a machine learning classification model using the K-Nearest Neighbors (KNN) algorithm on the Iris dataset. The model is trained to classify iris flowers into three different species based on their features.

## Features

* Load and explore the Iris dataset
* Data preprocessing and shuffling
* Train-test split
* KNN classification model
* Model evaluation using accuracy score
* Confusion Matrix visualization
* Heatmap visualization

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

## How to Run

1. Install the required libraries:

   ```bash
   pip install -r requirements.txt
   ```

2. Run the project:

   ```bash
   python project.py
   ```

## Output

The project displays:

* Model Accuracy
* Classification Results
* Confusion Matrix
* Heatmap Visualization
